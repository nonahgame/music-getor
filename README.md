# LyricBeats AI

Agentic music gen with LangChain/LangGraph.

## Local Setup
1. `pip install -e .`
2. Copy `.env.example` to `.env`, add keys.
3. `alembic upgrade head`
4. `python app.py`

## Deploy to Render
See instructions above.
